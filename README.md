# 2019-nCoV 酷Q插件
## 插件功能

- **全国疫情查询**
  指令：疫情动态
- **地区疫情查询**
  指令：地区+疫情 例：合肥疫情
- **其它国家疫情查询**
  指令：地区+全国疫情 例：泰国疫情
- **群实时推送最新疫情消息**
  指令：订阅疫情最新消息 取消：取消订阅疫情最新消息
- **新冠病毒谣言排行榜**
  指令：谣言排行榜
